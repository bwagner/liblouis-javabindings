Liblouis Java-Bindings
======================

This project provides Java bindings to the [liblouis](http://code.google.com/p/liblouis/) library. 

Prerequisite installs
------------------------

* [java](http://java.sun.com)
* [liblouis](http://code.google.com/p/liblouis/)
